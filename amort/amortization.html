Amortization Calculator or "Periodic Payment" coded by CW Coleman (Java)







Here is the eqaution I used to to calculate the peroiodic payment.
A = [ P ( r / n ) ] / [ 1 - ( 1 + r / n )^(-n*t)]         
A = Payment amount.
P = Amount borrowed.
r = Interest rate per period.
n = Number of payments per year (12 in this case).
t = Number of years.
